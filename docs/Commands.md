# Slash Commands

| Command | Effect |
|---|---|
| `/wcdt` | Toggle the CD tracker |
| `/wcdt show` | Show the tracker |
| `/wcdt hide` | Hide the tracker |
| `/wcdt settings` | Open the per-spell settings panel |
| `/wcdt reset` | Reset position to center of screen |

## In-header actions

- **Cog icon** — open settings.
- **Drag header** — move.
- **BOTTOMRIGHT bracket** — resize grip.
