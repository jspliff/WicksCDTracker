# Getting Started

## Install

- **CurseForge client** — search for "Wick's CD Tracker".
- **Manual** — download the latest ZIP from [CurseForge](https://www.curseforge.com/wow/addons/wicks-cd-tracker) or [GitHub Releases](https://github.com/jspliff/WicksCDTracker/releases), extract the `WicksCDTracker` folder into `World of Warcraft\_classic_\Interface\AddOns\`.

## First run

1. `/wcdt` — toggles the tracker window.
2. You'll see a row per party/raid member with that character's tracked cooldowns.
3. Class-colored names. Icons are ready (full color) or on-cooldown (greyed with timer overlay).

## Settings

Click the **cog** in the header's top-right corner to open settings.

- **Per-spell toggles** — enable/disable any tracked ability.
- **Interrupts only** — one click to strip everything except kicks. Useful on kick-critical progression pulls.

## Moving and resizing

- **Drag the header** to move.
- **Drag the BOTTOMRIGHT fel-green bracket** to resize (it's both the visual accent and the resize grip).

Position and size are saved per character.
