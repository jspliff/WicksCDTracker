# FAQ

### Which cooldowns does it track?

The defaults include kicks (all classes), Battle Rez, Soulstone, Bloodlust/Heroism, Pain Suppression, Innervate, Tricks of the Trade, Lay on Hands, Aspect of the Pack, Divine Shield, and more. Toggle any of them on/off in the settings cog.

### Can I add custom spells?

Not in the current release — planned. Tracked abilities are curated for raid utility.

### Does it work in PvP?

Yes — it shows party/raid members regardless of content type. It uses party/raid membership, not instance type.

### Does "Interrupts only" hide other classes' kicks too?

No — kicks from every class (Earth Shock, Counterspell, Pummel, Kick, Feral Charge, Silence, Strangulate, etc.) stay visible. Only non-interrupt abilities hide.

### Position reset didn't work

Try `/wcdt reset` then `/reload`.

### Is this for Era / Season of Discovery / Retail?

TBC Classic only (Interface 20505).
